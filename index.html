CH. 3 / INSTALLATION

/ 003

Terms. Free. Forever. No accounts, no license keys, no telemetry. One cat per desk recommended; not enforced.

MACOS — NO SECURITY WARNINGS

Paste this in Terminal. It builds BYTECAT.app in your Applications folder and launches it — no Gatekeeper popups, because the app is assembled on your own machine and never carries the browser's quarantine flag.

curl -fsSL https://ranch6.github.io/bytecat/downloads/install.sh | sh

Needs Python 3 with Tk — the installer checks and tells you if it's missing (brew install python-tk). Afterwards BYTECAT lives in ~/Applications; open it from Spotlight like any app. Uninstall by deleting the app.

WINDOWS / LINUX / MANUAL

One small zip: the cat, double-click launchers, and a plain-text README.

DOWNLOAD BYTECAT.ZIP ▸

Unzip → double-click BYTECAT. Windows installs from python.org include Tk. Linux: sudo apt install python3-tk. Prefer the bare file? bytecat.py
